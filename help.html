写真に位置情報を含める設定方法

スマートフォンのカメラで位置情報（GPS）を写真に記録するための基本設定をまとめました。

共通の前提

  • 端末の位置情報（GPS）機能がオンになっていること。
  • 撮影後に位置情報は自動では付与されないため、設定は撮影前に有効化する。

iPhone の場合

  1. 「設定」アプリを開く。
  2. 「プライバシーとセキュリティ」→「位置情報サービス」をオンにする。
  3. アプリ一覧から「カメラ」を選択し、「アプリの使用中のみ許可」など位置情報アクセスを許可する（表示によっては「撮影場所を保存」等の表記になる場合があります）。
  4. カメラアプリで撮影すると、位置情報が写真に保存される。

Android の場合

  1. 「設定」アプリで「位置情報」または「セキュリティと位置情報」を開き、位置情報をオンにする。
  2. カメラアプリを起動し、設定アイコン（歯車）などから「位置情報を保存」「撮影場所を保存」「ジオタグ」等の項目をオンにする。
  3. 以後の撮影で、位置情報が写真に記録される。

※メニュー名は端末やOSのバージョンによって異なる場合があります。

iPhoneで位置情報が読み取れないとき

  1. 設定 → カメラを開き、フォーマットを「高効率」、位置情報はオンにする。
  2. 設定 → プライバシーとセキュリティ → 位置情報サービス → カメラで、許可を「アプリの使用中」に設定する。
  3. 設定 → （使用ブラウザ） → 写真でアクセス権を「プライベートアクセス」にし、共有時の「オプション → 位置情報」が「含める」になっていることを確認する。
  4. 同じ「オプション」でフォーマットを「現在のフォーマット」に設定する（HEICのまま共有され、メタ情報が保持されます）。
  5. ブラウザのファイル選択画面から直接撮影すると位置情報が削除される場合があるため、先にカメラアプリで撮影して写真ライブラリから選択する。

アプリに戻る

位置情報の取り扱いには注意しましょう。
